# E-commerce Sales Analysis Dashboard

# Overview

This project analyzes e-commerce sales data using SQL, Python, and Power BI to generate business insights and visualize sales performance.

# Tools & Technologies

* SQL
* Python (Pandas, NumPy, Matplotlib)
* Power BI

# Project Workflow

1. Data Cleaning and Preparation
2. SQL Data Analysis
3. Exploratory Data Analysis using Python
4. Dashboard Creation in Power BI

# Key Insights

* Technology category generates highest profit
* West region contributes the most to overall sales
* Sales peak during November and December
* Furniture category has lower profitability
